Drag Gesture Demo - Visual Impact (elementVisuallyImpacted)

This demo demonstrates the advanced elementVisuallyImpacted feature where:

  • Blue handle: The element you grab and drag from
  • Red container: The element that physically moves (elementToImpact)
  • Orange inner element: The element used for visual calculations (elementVisuallyImpacted)
  • Key concept: Constraints and auto-scroll are calculated based on the inner element's position, not the container
  • Real-world use case: Moving a table column where you grab the header handle, move the entire column container, but track a specific cell for collision detection

Drag Flow

Grab Handle → Move Container → Track Inner Element → Check Obstacles
