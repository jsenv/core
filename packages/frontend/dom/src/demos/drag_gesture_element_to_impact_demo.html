Drag Gesture Demo - elementToImpact

This demo demonstrates the elementToImpact feature where you can drag from one element but actually move a different element:

  • Blue handles: The elements you grab and drag from
  • Red targets: The elements that actually move when you drag the handles
  • Gray obstacles: Static elements that provide collision constraints
  • Key concept: Obstacles and frontiers are calculated based on the target element's position, not the handle
  • Try it: Drag the blue handles and watch how the red targets move while respecting obstacles
