Drag Reference Element Demo

This demo tests drag operations where the reference element (what you click/drag) is different from the element to move (what actually moves).

Reference Element (click to drag)
Element to Move (follows the drag)

Test Scenarios:

  • Case 1: Both elements in document - Reference element controls movement of another element in the same coordinate system
  • Case 2: Reference in document, Element to move in scroll container - Cross-coordinate system drag operation
  • Case 3: Both elements in scroll container - Reference element and target both within scrollable content
  • Case 4: Reference in scroll container, Element to move in document - Reverse cross-coordinate scenario
