# [addAll registers multiple keys at once](../../i18n.test.js)

```js
const i18n = createI18n({ runtimeLang: "fr" });
i18n.addAll({
  "one minute": { fr: "une minute" },
  "one person": { fr: "une personne" },
});
return {
  minute: i18n("one minute"),
  person: i18n("one person"),
};
```

  "person": "une personne"
